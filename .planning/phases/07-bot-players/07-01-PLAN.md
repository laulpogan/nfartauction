  <behavior>
    - chooseBotCard: given a hand with multiple artists, conservative picks the artist with highest cumulative roundValues; aggressive picks the artist with highest current-round count (trending); erratic picks randomly (seeded)
    - chooseBotBid (open): returns currentBid + increment scaled by personality valuation; returns null if bid would exceed money * passThreshold
    - chooseBotBid (once_around): returns a single bid based on perceived value (roundValues for the artist + personality multiplier + noise); null to pass
    - chooseBotBid (sealed_bid): similar to once_around but with wider personality-scaled variance
    - chooseBotBid (fixed_price): returns true (accept) if fixedPrice < perceived value * personality threshold; false (pass) otherwise
    - chooseBotBid (double / waiting_second): returns a matching Card from hand if available, else null (pass)
    - chooseBotSlots: returns SIM_CONFIG.SLOTS_PER_DAY slots weighted by BOT_SLOT_WEIGHTS, with neighborhoods from BOT_NEIGHBORHOOD_PREFS
    - All functions accept a `random` number (0-1) parameter for deterministic testing
  </behavior>
  <action>
1. Create `src/lib/bot-engine.ts` with these pure functions:

   **`perceiveArtistValue(artist, game): number`** (internal helper)
   - Base value = game.roundValues[artist] (cumulative value from past rounds)
   - Current round demand = game.artistCounts[artist] * 5000 (more paintings = more demand signal)
   - Return base + demand

   **`chooseBotCard(hand, game, personality, random): Card`**
   - If hand is empty, throw (caller must guard)
   - Conservative: sort hand by perceiveArtistValue descending, pick first (play highest value artist)
   - Aggressive: sort hand by game.artistCounts[card.artist] descending (chase trending), pick first
   - Erratic: pick hand[Math.floor(random * hand.length)]
   - If the chosen card is a 'double' type, check if any other card in hand has the same artist — prefer non-double if available (avoid getting stuck waiting for second card as a bot)

   **`chooseBotBid(auction, game, personality, money, random): number | null`**
   - Dispatch by auction.auctionType:
   - **open**: perceived = perceiveArtistValue(auction.cards[0].artist, game); maxBid = perceived * BOT_CONFIG.valuationMultiplier[personality]; noise = (random - 0.5) * 2 * BOT_CONFIG.bidNoiseRange[personality] * perceived; bid = Math.floor(auction.currentBid + 1000 + noise). Return null if bid > money * BOT_CONFIG.passThreshold[personality] or bid > maxBid. Return bid clamped to at least currentBid + 1000.
   - **once_around**: perceived value * multiplier + noise. Return null if > money * passThreshold. Return Math.floor(bid) or null.
   - **sealed_bid**: same as once_around but noise range doubled (wider variance).
   - **fixed_price**: if auction.fixedPrice <= perceived * multiplier * (1 + noise*0.1), return auction.fixedPrice (accept). Else return null (pass).
   - **double**: return null (bots don't initiate bids on double auctions — they participate via chooseBotSecondCard)

   **`chooseBotSecondCard(hand, auction, personality, random): Card | null`**
   - Find cards in hand matching auction.cards[0].artist
   - If found, return one (conservative picks the non-double type if available; erratic picks randomly)
   - If no match, return null (pass)

   **`chooseBotSlots(playerSim, game, personality, random): TimeSlot[]`**
   - Generate SIM_CONFIG.SLOTS_PER_DAY slots
   - For each slot: weighted random selection from SlotType using BOT_SLOT_WEIGHTS[personality]
   - Neighborhood: pick from BOT_NEIGHBORHOOD_PREFS[personality] using random
   - Each slot gets a generated id: `bot-slot-${i}`
   - Return the TimeSlot array

   All functions must be pure: no Math.random, no Date.now, no console. The `random` parameter provides all entropy.

2. Create `src/lib/bot-engine.test.ts` (Vitest) covering:

   - chooseBotBid (once_around): returns number or null
   - chooseBotBid (sealed_bid): returns number with wider variance than once_around
   - chooseBotBid (fixed_price): accepts when price is low, passes when high
   - chooseBotSecondCard: returns matching card or null
   - chooseBotSlots: returns correct number of slots; conservative favors sleep/gallery_work; aggressive favors party/art_fair
   - Test with fixed random values (0.0, 0.5, 0.99) for determinism
